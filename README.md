# Co2-Meter-Arduino

This project was made as a school project by students of the vocational school BBS-Haarentor.

The following components are needed:

  1. Arduino Uno Rev 3
  2. BENSN 9V 1A Power supply
  3. ARD SensorKit Arduino
  4. KEYSTUDIO CCS811
